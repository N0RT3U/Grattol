# Grattol CIS 네일시장 대응 프로젝트 통합 보고서

## 1. 프로젝트 개요

본 프로젝트는 독일 네일케어 브랜드 `Grattol`이 2020년 3월 전후 러시아·CIS 시장에서 맞닥뜨릴 수요 충격에 대응하기 위해, 소비자 행동 로그를 기반으로 시장 구조를 재해석하고 실행 가능한 대시보드와 자동화 운영 체계를 설계한 데이터 분석 프로젝트다. 핵심 문제의식은 세 가지다.

첫째, 러시아 네일 시장은 코로나19 확산, 유가 급락, 루블화 약세, 락다운 조치가 겹치며 오프라인 B2B 네일 살롱 수요가 급감할 위험이 컸다. 둘째, 시장 1위 경쟁사 `Runail`은 저가 물량 공세로 우위를 점하고 있었으나, `HEMA` 관련 안전성 이슈가 존재했다. 셋째, `Grattol`은 안전성과 사용성 측면의 강점을 보유하고 있었지만, 정작 구매가 일어나는 시간대와 요일, 그리고 고객 세그먼트별 행동 패턴에 맞춘 정밀한 실행 체계는 부족했다.

이에 본 프로젝트는 단순 현황 요약을 넘어 다음 세 가지를 동시에 수행했다.

1. 시장·경쟁 구조를 계량적으로 파악하는 탐색적 분석
2. `RFM + KMeans` 기반 고객 세그먼트 정의와 이탈 위험 진단
3. 운영·마케팅·CRM 팀이 즉시 활용할 수 있는 대시보드 및 전처리 자동화 설계

## 2. 데이터 개요와 전처리

### 2.1 데이터 출처와 분석 범위

프로젝트 발표 대본 기준 원천 데이터는 2019년 10월부터 2020년 2월까지의 소비자 행동 로그 약 2천만 건이다. 실제 저장소에는 용도별로 세 단계의 분석 데이터가 존재한다.

- `final_full.csv` : 상위 브랜드 전체 시장 분석용 통합 데이터
- `grattol_runail.csv` : `Grattol`과 `Runail` 경쟁 비교용 데이터
- `nail_market_v3.csv` : `Grattol` 단독 심층 분석 및 세그먼트 분석용 최종 데이터

저장소 기준 데이터 규모는 다음과 같다.

| 구분 | 행 수 | 주요 용도 |
| --- | ---: | --- |
| `final_full.csv` | 5,702,963 | 전체 시장 KPI 및 대시보드 메인 화면 |
| `grattol_runail.csv` | 2,381,492 | 경쟁사 비교, 퍼널 비교 |
| `nail_market_v3.csv` | 852,079 | Grattol 심층 분석, 세그먼트, CRM |

`nail_market_v3.csv` 기준 유니크 사용자 수는 123,405명이며, 구매 고객은 17,146명이다.

### 2.2 전처리 절차

전처리 자동화 스크립트와 노트북을 종합하면 파이프라인은 다음 순서로 구성된다.

1. 월별 CSV 파일 적재 및 병합
2. `price <= 0` 이상치 제거
3. 타겟 브랜드 및 상위 브랜드 필터링
4. `event_time`을 `Europe/Moscow` 기준으로 변환
5. 분석 기준 시점을 `2020-02-29 23:59:59`로 고정
6. 필요 시 클러스터 결과 파일을 `MERGE_KEYS` 기준으로 병합
7. `event_date`, `event_hour` 등 파생 변수 생성
8. 대시보드 반영용 `clear.csv` 혹은 최종 분석 파일 생성

즉, 본 프로젝트의 전처리는 단순 정제 수준이 아니라, 시계열 분석과 세션 기반 퍼널 분석, 그리고 고객 세그먼트 병합이 가능한 분석형 데이터셋을 만드는 과정이었다.

## 3. 분석 설계

본 프로젝트의 분석은 네 개의 계층으로 설계되었다.

### 3.1 시장·행동 탐색(EDA)

- 전체 이벤트 흐름과 시계열 수요 변화 확인
- 요일·시간대별 행동 히트맵 분석
- `Grattol`과 `Runail`의 가격 포지셔닝 비교
- 퍼널 전환율, 재구매율, 코호트 리텐션 분석
- 평균 대비 시간대·요일별 성과 약점 도출

### 3.2 경쟁 우위 해석

- `Runail`은 저가·대량 판매 구조인지
- `Grattol`은 프리미엄 가격대 집중형인지
- 언제 `Runail`에 밀리고, 어디에서 `Grattol`의 강점이 살아나는지 계량화

### 3.3 고객 세그먼트 모델링

- 구매 고객 기준 `RFM + Basket Size` 설계
- `MiniBatchKMeans`와 `KMeans`를 통해 5개 군집 도출
- 군집별 구매력, 최근성, 바스켓 크기, 이탈 위험 해석

### 3.4 실행 시스템 설계

- 운영·기획용 메인 대시보드
- 마케팅용 고객 동향 대시보드
- CRM용 세그먼트 대시보드
- 월별 파일만 넣으면 갱신 가능한 전처리 자동화 프로그램

## 4. 핵심 분석 결과

### 4.1 시장 수요와 행동 패턴

탐색 결과, 수요는 11월 중순 이후 급격히 상승했고 1월 초 신정 및 동방정교회 성탄절 연휴 구간에서 크게 둔화되었다. 이후 2월 말로 갈수록 수요가 다시 약해지는 모습이 관찰되며, 이는 코로나19 확산 전조와 맞물린다.

행동 히트맵에서는 두 가지 패턴이 명확했다.

- 탐색은 밤 10시~11시에 집중된다.
- 실제 구매는 평일 오후 2시~4시에 상대적으로 많이 발생한다.

즉, 고객은 밤에 탐색하고 낮에 결제한다. 따라서 광고 집행과 전환 프로모션을 같은 시간대에 두는 방식은 비효율적이며, 인지 제고와 구매 전환을 분리 운영해야 한다.

### 4.2 경쟁사 대비 가격 포지셔닝

`grattol_runail.csv` 구매 기준 재계산 결과는 다음과 같다.

| 지표 | Grattol | Runail |
| --- | ---: | ---: |
| 평균 상품 가격 | 5.38달러 | 3.08달러 |
| 중앙값 상품 가격 | 5.24달러 | 2.38달러 |
| 4~7달러 가격대 구매 비중 | 88.52% | 11.70% |
| 주문당 평균 매출(AOV) | 12.67달러 | 7.57달러 |
| 구매 고객 1인당 매출(ARPPU) | 15.53달러 | 9.70달러 |

이 결과는 `Runail`이 저가·박리다매 구조인 반면, `Grattol`은 4~7달러 구간에 강하게 집중된 합리적 프리미엄 포지셔닝을 갖고 있음을 보여준다. 즉, `Grattol`의 전략은 가격 경쟁에서 정면 승부하기보다, 안전성과 사용경험을 결합한 프리미엄 홈케어 브랜드로 차별화하는 방향이 타당하다.

### 4.3 퍼널 전환율 비교

브랜드별 퍼널 전환율은 세션 기준으로 계산되었다.

- `View → Cart 전환율 = cart 세션 수 / view 세션 수`
- `View → Purchase 전환율 = purchase 세션 수 / view 세션 수`
- `Cart → Purchase 전환율 = purchase 세션 수 / cart 세션 수`

재계산 결과는 다음과 같다.

| 전환 지표 | Grattol | Runail |
| --- | ---: | ---: |
| View → Cart | 36.08% | 43.69% |
| View → Purchase | 9.47% | 12.22% |
| Cart → Purchase | 26.24~26.25% | 27.98% |

분모와 분자는 `distinct user_session` 기준이다. Grattol 단독 데이터 기준 실제 세션 수는 아래와 같다.

- View 세션: 221,942
- Cart 세션: 80,070
- Purchase 세션: 21,018

따라서

- `View → Cart = 80,070 / 221,942 = 36.08%`
- `View → Purchase = 21,018 / 221,942 = 9.47%`
- `Cart → Purchase = 21,018 / 80,070 = 26.25%`

즉, `Grattol`은 장바구니 진입과 최종 구매 모두에서 `Runail`보다 약하다. 이는 제품 매력 부족이라기보다, 결정적 시간대와 구매 직전 설득 장치가 부족했음을 시사한다.

### 4.4 재구매율과 리텐션

재구매율은 `다른 날짜에 2회 이상 구매한 고객 / 전체 구매 고객`으로 계산했다.

- `Grattol`: 15.41~15.42%
- `Runail`: 17.23~17.24%

수식은 다음과 같다.

1. 구매 로그에서 `purchase_date`를 일 단위로 생성
2. 브랜드·유저별 구매 일수(`purchase_days`) 계산
3. `purchase_days >= 2` 이면 재구매 고객으로 분류
4. 재구매율 = 재구매 고객 비율

주간 코호트 리텐션과 세션 기준 재방문 분석에서는, 연휴 직후 재방문률이 다시 높아지는 흐름과 함께 `Grattol`이 구매 이후 지속 탐색을 유도하는 브랜드라는 정성적 해석이 가능했다. 다만 대본에서 강조한 것처럼, 코로나19 락다운이 시작되면 오프라인 살롱 기반 고객의 반복구매 구조는 훼손될 가능성이 크다.

### 4.5 시간대·요일별 약점

평균 대비 성과(`lift`) 분석 로직은 다음과 같다.

- 전체 평균 `Grattol` 점유율을 `baseline_share`로 정의
- 가격대/시간대/요일별 `Grattol` 매출 점유율을 계산
- `lift = 구간별 Grattol 점유율 - baseline_share`

노트북 해석 결과는 일관적이다.

- 구매 결정이 많이 일어나는 오후 2~4시 구간에서 `Runail`에 밀린다.
- 수요일 성과가 특히 약하다.
- 따라서 `수요일 + 14~16시`는 `Grattol`의 핵심 보완 타이밍이다.

이 결과는 발표 대본의 `타임세일`, `수요일 프로모션`, `밤 22~23시 집중 광고` 제안의 직접적 근거다.

## 5. 고객 세그먼트 분석

### 5.1 RFM + Basket Size 설계

구매 고객에 대해 다음 네 변수를 설계했다.

- `Recency = 분석 기준일 - 마지막 구매일`
- `Frequency = 구매 이벤트 수`
- `Monetary = 구매 금액 합계`
- `Basket Size = 세션별 구매 상품 수의 고객 평균`

구현 절차는 다음과 같다.

1. `purchase` 이벤트 중 `brand == grattol`만 추출
2. 유저·세션별 상품 수를 집계하여 `items_in_basket` 생성
3. 유저별 평균 장바구니 크기를 `basket_size`로 계산
4. 유저별 `last_purchase`, `frequency`, `monetary` 집계
5. `frequency`, `monetary`, `basket_size`에 `log1p` 적용 후 표준화
6. `MiniBatchKMeans`와 `KMeans`로 군집화 수행

### 5.2 군집 수 선택과 검증

군집 수는 `k = 2 ~ 8` 범위를 탐색한 뒤 `k = 5`로 확정했다. 최종 평가 지표는 다음과 같다.

- Silhouette Score: 0.4005
- Calinski-Harabasz: 14558.41
- Davies-Bouldin: 0.8760
- Inertia(SSE): 15596.75

실무형 세그먼트 목적을 감안하면 설명 가능한 수준의 분리가 확보되었다고 볼 수 있다.

### 5.3 세그먼트 프로파일

최종 5개 군집의 평균 프로파일은 아래와 같다.

| 세그먼트 | Recency | Frequency | Monetary | Basket Size | 해석 |
| --- | ---: | ---: | ---: | ---: | --- |
| VIP salon | 64.6 | 13.62 | 72.38 | 9.80 | 초고가·고빈도·대바스켓 |
| Core repeaters | 30.6 | 5.09 | 28.11 | 3.38 | 재구매 중심 핵심 고객 |
| At-risk | 102.7 | 3.08 | 16.69 | 2.79 | 구매력은 있으나 오래 안 산 고객 |
| New/light buyers | 30.0 | 1.48 | 8.04 | 1.31 | 최근 유입 라이트 고객 |
| Inactive/one-time | 98.6 | 1.06 | 5.30 | 1.02 | 장기 이탈·일회성 고객 |

태그 부여 로직은 다음과 같다.

- `VIP salon`: `frequency + monetary + basket_size` 종합 순위 최상위 군집
- `Core repeaters`: 최근성이 좋고 구매력도 중간 이상
- `New/light buyers`: 최근성이 좋지만 구매력은 낮음
- `At-risk`: 최근성은 나쁘지만 구매력은 존재
- `Inactive/one-time`: 최근성도 나쁘고 구매력도 낮음

### 5.4 VIP 집중도와 매출 구조

최종 `Grattol` 단독 분석 파일 기준 세그먼트별 구매자 수는 17,146명이며, 그중 `VIP salon`은 1,010명이다.

- `VIP 비중 = 1,010 / 17,146 = 5.89%`

세그먼트별 매출 구조는 다음과 같다.

| 세그먼트 | 구매자 비중 | 매출 비중 |
| --- | ---: | ---: |
| VIP salon | 5.89% | 27.47% |
| Core repeaters | 13.85% | 25.08% |
| At-risk | 20.17% | 21.69% |
| New/light buyers | 29.84% | 15.45% |
| Inactive/one-time | 30.25% | 10.32% |

즉, `VIP salon + Core repeaters`는 구매자 기준 19.74%지만 매출 기준 52.54%를 차지한다. 발표 대본에서는 이 조합을 `매출 60% 이상`으로 설명했으나, 저장소의 최종 분석 파일 기준 재계산 결과는 약 52.5%다. 보고서 기준 공식 수치는 재계산값을 따른다.

또한 `VIP salon`의 1인당 지표는 다음과 같다.

- `revenue_per_buyer = 73,104.88 / 1,010 = 72.38`
- `purchases_per_buyer = 13,755 / 1,010 = 13.62`

반면 비VIP는

- `revenue_per_buyer = 193,051.92 / 16,136 = 11.96`
- `purchases_per_buyer = 35,762 / 16,136 = 2.22`

즉, 소수의 B2B형 헤비 고객에 대한 의존도가 매우 높다. 락다운 시점에서 이 구조는 강점이 아니라 리스크다.

### 5.5 장바구니 이탈 분석

노트북은 `95% 구매 결정이 장바구니 투입 후 572.7시간 안에 이루어진다`고 계산했다. 이는 약 23.9일이다. 다만 실제 운영 지표는 단순화된 `21일(3주)` 윈도우를 사용했다.

엄격한 장바구니 이탈(`abandon_strict_21d`) 정의는 다음과 같다.

- 21일 내 `remove_from_cart`가 발생하고
- 21일 내 구매가 없으며
- 동일 세션 내 구매도 없는 경우

즉,

`abandon_strict_21d = remove_in_21d ∧ not purchase_in_21d ∧ not purchase_same_session`

전체 Grattol 기준 엄격 이탈률은 다음과 같다.

- `42.69%`

세그먼트별 결과는 다음과 같다.

| 세그먼트 | carts | strict abandon rate |
| --- | ---: | ---: |
| New/light buyers | 30,193 | 45.57% |
| Inactive/one-time | 21,217 | 44.69% |
| At-risk | 37,325 | 42.84% |
| Core repeaters | 57,016 | 39.85% |
| VIP salon | 63,621 | 29.90% |

이 결과는 세그먼트별 CRM 전략의 우선순위를 명확히 보여준다. 신규·비활성 고객은 장바구니 단계에서 가장 많이 이탈하므로 쿠폰, 리마인드, 한정 할인, 묶음 추천이 필요하다. 반면 VIP는 구매 전환율이 높기 때문에 혜택의 초점이 `방어와 유지`에 맞춰져야 한다.

## 6. 시계열 예측과 수요 해석

머신러닝 노트북에서는 시계열 회귀를 통해 3월 매출을 예측했고, `미세한 하향 횡보`가 관찰되었다. 노트북 메모에 따르면 최선 모델에서도 `R²는 여전히 음수`였으나, `MAPE`, `MAE`, `RMSE`는 개선되었다. 이는 단순한 계절성만으로는 2020년 3월 이후의 급격한 외생 변수 충격을 설명하기 어렵다는 의미다.

또 다른 회귀 실험에서는 일별 판매량 예측을 위해 `RandomForestRegressor`를 사용했다. 변수 중요도 해석은 아래와 같다.

- `roll_7(최근 7일 평균 판매량)` 중요도 약 0.79
- `lag_1(전일 판매량)`이 그 다음
- `day`, `dow`, `month`는 보조적 역할

즉, 판매량은 장기 구조 변화보다는 최근 추세에 가장 크게 좌우된다. 이는 외생 충격 직전의 흐름을 빠르게 감지하는 운영형 모니터링이 필요하다는 점을 뒷받침한다.

## 7. 대시보드 설계와 KPI 정의

### 7.1 메인 대시보드

메인 대시보드는 운영·기획 팀을 위한 전체 현황판으로 설계되었다. 상단 KPI는 다음 항목으로 구성된다.

- 전체 매출액
- 목표 대비 실적
- 총 유저 수
- 구매 전환율

전체 시장 파일(`final_full.csv`) 기준 재계산 결과는 다음과 같다.

- 총 유저 수: 551,360명
- 누적 매출: 1,640,368.16달러

이는 발표 대본의 `55만 명`, `164만 달러`와 일치한다.

### 7.2 마케팅 대시보드

마케팅 대시보드는 고객 동향과 프로모션 대응을 위한 화면이다. 다만 기획 문서에는 `1인당 매출수`, `객단가`, `평균주문금액`이 일부 중복 표현되어 있어 본 보고서에서는 아래처럼 표준화한다.

- `누적매출 = Σ price, event_type = purchase`
- `활성 구매 유저 = distinct user_id, event_type = purchase`
- `구매전환율(유저 기준) = 구매 유저 수 / 전체 유저 수`
- `ARPPU = 누적매출 / 활성 구매 유저`
- `총주문수 = distinct user_session, event_type = purchase`
- `AOV = 누적매출 / 총주문수`
- `총상품수 = purchase 행 수`
- `주문당 평균 상품 수 = 총상품수 / 총주문수`
- `재구매율 = 2일 이상 구매 고객 수 / 전체 구매 고객 수`

`nail_market_v3.csv` 기준 재계산 값은 다음과 같다.

| KPI | 식 | 값 |
| --- | --- | ---: |
| 총 유저 수 | `distinct user_id` | 123,405 |
| 구매 유저 수 | `distinct user_id where purchase` | 17,146 |
| 구매전환율 | `17,146 / 123,405` | 13.89% |
| 누적매출 | `Σ price where purchase` | 266,156.80달러 |
| 총주문수 | `distinct purchase user_session` | 21,018 |
| 총상품수 | `count(purchase rows)` | 49,517 |
| ARPPU | `266,156.80 / 17,146` | 15.52달러 |
| AOV | `266,156.80 / 21,018` | 12.66달러 |
| 주문당 평균 상품 수 | `49,517 / 21,018` | 2.36개 |
| 재구매율 | `repeat buyers / buyers` | 15.41% |

### 7.3 CRM 세그먼트 대시보드

CRM 대시보드는 다음 지표를 중심으로 설계되었다.

- 세그먼트 비중
- 세그먼트별 매출 추이
- 세그먼트별 장바구니→구매 전환율
- 세그먼트별 평균 구매 가격대
- 세그먼트별 평균 구매 수량

이 설계는 단순한 `누가 VIP인가`를 보여주는 데 그치지 않는다. 어떤 고객군에서 이탈 신호가 강한지, 어떤 군집에서 유지 비용 대비 효과가 높은지, 어느 군집에서 홈케어 전환 패키지를 우선 제안해야 하는지를 실행 단위로 연결한다.

## 8. 실행 전략 제안

분석 결과를 실제 액션으로 전환하면 다음과 같다.

### 8.1 탐색 시간대 광고 집중

고객은 밤 22~23시에 많이 탐색한다. 이 시간대에는 브랜드 광고와 안전성 메시지, 홈케어 사용성 콘텐츠를 집중 노출해야 한다. 목적은 즉시 구매가 아니라 인지 확보다.

### 8.2 구매 결정 시간대 타임세일

실제 구매는 평일 오후 14~16시에 집중된다. 그러나 이 시간대에 `Grattol`은 `Runail`보다 약하다. 따라서 해당 구간에는 타임세일, 무료배송 임계값 조정, 장바구니 쿠폰 발급 같은 전환형 프로모션을 배치해야 한다.

### 8.3 수요일 집중 개선

요일별 lift 분석 결과 수요일이 구조적 약점으로 해석된다. 수요일은 경쟁사 회수 방어용 요일로 설정하고, 제한시간 할인이나 세트 구성 노출을 우선 배치할 필요가 있다.

### 8.4 장바구니 이탈 방지

엄격한 21일 이탈률이 42.69%이며 신규·비활성 고객은 45% 내외까지 올라간다. 따라서

- 장바구니 보유 1~3일차 리마인드
- 7일차 가격저항 완화 쿠폰
- 14일차 홈케어 세트 추천
- 21일차 전환 실패 고객 재활성 캠페인

과 같이 단계형 CRM 흐름을 설계하는 것이 타당하다.

### 8.5 VIP salon 방어 전략

VIP는 전체 구매자의 5.89%지만 매출의 27.47%를 차지한다. 다만 락다운 시기에는 오프라인 살롱 수요가 줄 가능성이 높다. 따라서 `대량 구매 혜택`을 유지하되, 홈케어 키트·소포장 세트로 전환 가능한 상품 구성을 별도로 제안해야 한다.

## 9. 자동화 운영 체계

프로젝트는 분석 결과만 제시한 것이 아니라, 월별 데이터 갱신을 위한 전처리 자동화 프로그램까지 설계했다. 스크립트 기준 동작 방식은 다음과 같다.

1. 월별 CSV 파일 또는 폴더 경로 입력
2. 필요 시 클러스터 파일 경로 입력
3. 타임존 변환, 필터링, 파생변수 생성 자동 수행
4. `clear.csv` 저장
5. Tableau/Prep 연동 데이터로 즉시 사용

즉, 이 프로젝트는 일회성 분석물이 아니라, 데이터가 매월 들어와도 운영팀이 직접 갱신 가능한 준운영 시스템에 가깝다.

## 10. 한계와 검증 메모

본 프로젝트는 분명한 실행 가치를 가지지만, 다음 한계를 동시에 인식해야 한다.

1. 분석 기간이 2019년 10월~2020년 2월로 짧아 장기 계절성 검증에는 제약이 있다.
2. 팬데믹, 환율, 유가, 락다운 같은 외생 변수는 직접 피처로 연동되지 않았다.
3. 노트북의 일부 발표 수치와 최종 분석 파일 재계산 결과는 완전히 일치하지 않는다.

특히 검증 메모는 다음과 같다.

- `VIP + Core repeaters가 매출 60% 이상`이라는 발표 문구는 최종 파일 재계산 시 `52.54%`로 확인된다.
- `3주 안에 구매 여부 결정`이라는 발표 서술은 노트북 계산상 `95% 구매 결정 시간 = 572.7시간(약 23.9일)`과 유사하지만 완전히 동일하지는 않다. 실무 지표는 21일 윈도우를 사용했다.
- 메인 대시보드의 `55만 명`, `164만 달러`는 전체 시장 파일 기준 수치이고, `13.89% 전환율`, `266.2K 달러` 등은 Grattol 단독 파일 기준 수치다. 따라서 KPI를 사용할 때는 분석 범위를 명확히 구분해야 한다.

이 불일치를 보고서에서 명시하는 이유는, 프로젝트의 신뢰도를 높이기 위해서다. 발표용 메시지와 저장소 기준 재현 수치를 분리하는 것이 기술 보고서로서는 더 타당하다.

## 11. 결론

본 프로젝트는 `Grattol`의 강점이 단순히 제품 품질에 있는 것이 아니라, `안전한 프리미엄 홈케어`라는 명확한 포지셔닝에 있음을 데이터로 보여주었다. 동시에 `Runail` 대비 낮은 전환율, 오후 2~4시와 수요일의 약세, 장바구니 이탈률 42.69%, 그리고 VIP 의존적 매출 구조라는 리스크도 드러냈다.

정리하면 본 프로젝트의 성과는 세 가지다.

첫째, 시장과 경쟁 구조를 정량적으로 해석했다. 둘째, 고객을 행동 기반 세그먼트로 나누어 CRM 우선순위를 만들었다. 셋째, 이 결과를 대시보드와 자동화 프로그램으로 연결해 현업이 바로 사용할 수 있는 운영 체계로 변환했다.

따라서 이 프로젝트는 단순한 EDA 발표가 아니라, 위기 상황에서 `Grattol`이 무엇을 언제 누구에게 실행해야 하는지를 수치로 정리한 데이터 기반 의사결정 프레임워크라고 평가할 수 있다.
